# 🤖 Цифровые Отморозки — AI-команда для предпринимателя

> 6 AI-сотрудников в Telegram. Контент, аналитика, стратегия, дизайн, видео — из одной системы.

## Состав команды

| Агент | Роль | Что делает |
|-------|------|------------|
| 🧠 **Синсей** | Стратег | Принимает задачи, координирует команду, строит планы |
| ✍️ **Алёна** | Копирайтер | Пишет посты, статьи, тексты в голосе владельца |
| 🔍 **Критик** | Редактор | Проверяет контент перед публикацией |
| 🎨 **Маша** | Дизайнер | Карусели, обложки, визуальный контент |
| 📊 **Максим** | Аналитик | Тренды, SwipeFile, анализ конкурентов |
| 🎬 **Алекс** | Видео продюсер | Сценарии Reels, YouTube, монтаж |
| ⚙️ **Решала** | CRM-бот | Встречает клиентов, собирает заявки |

## Технология

Построено на **OpenClaw** — платформе для развёртывания AI-агентов в Telegram и других мессенджерах.

- Каждый агент — отдельный Telegram-бот со своей личностью
- Агенты общаются между собой (Синсей → Алёна → Критик → пользователь)
- Память хранится в файлах — агенты помнят всё между сессиями
- Голос пользователя зашит в `brand/` — контент звучит как ты

```
  maxim/       — 📊 Аналитик
  alex/        — 🎬 Видео продюсер
  reshala/     — ⚙️ CRM-бот (Решала)
shared/
  brand/       — профиль, голос, аудитория (настраивается под тебя)
  skills-list/ — список устанавливаемых скиллов
docs/
  INSTALL.md   — полная инструкция установки
  CONFIG.md    — настройка openclaw.json
  CUSTOMIZE.md — как настроить под свой бизнес
```

## Быстрый старт

```bash
# 1. Установи OpenClaw
npm install -g openclaw

# 2. Клонируй репо
git clone https://github.com/YOUR/otmorozki.git
cd otmorozki

# 3. Следуй инструкции
cat docs/INSTALL.md
```
