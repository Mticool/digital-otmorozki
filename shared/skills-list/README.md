# Скиллы — что устанавливать

Все скиллы доступны через `clawhub install <имя>` или на [clawhub.com](https://clawhub.com).

## Обязательные (для работы команды)

```bash
clawhub install copywriting       # метод скользкой горки, структура постов
clawhub install editing           # инфостиль + убеждающий копирайтинг
clawhub install headlines         # заголовки и хуки
clawhub install storytelling      # истории, кейсы, трёхактная структура
clawhub install selling-meanings  # продающие смыслы, офферы
clawhub install shared-learnings  # общие паттерны для всех агентов
clawhub install telegram          # посты для Telegram-канала
```

## Контент (по платформам)

```bash
clawhub install carousel          # Instagram карусели
clawhub install reels             # Reels/Shorts/TikTok
clawhub install youtube           # YouTube сценарии
clawhub install threads           # посты для Threads
```

## Визуал

```bash
clawhub install nano-banana       # генерация изображений
clawhub install heygen            # AI-аватары и видео
clawhub install veo-video         # генерация видео (Veo)
```

## Аналитика и исследования

```bash
clawhub install swipefile         # разбор видео/контента
clawhub install competitor-analysis  # анализ конкурентов
clawhub install apify-ultimate-scraper  # парсинг соцсетей
clawhub install instagram-research  # анализ Instagram
clawhub install news-summary      # мониторинг новостей
clawhub install seo-blog-writer   # SEO-статьи для блога
```

## Дополнительно

```bash
clawhub install prompt-engineer   # создание промптов
clawhub install personal-unpacking  # распаковка личности
clawhub install customer-research  # распаковка смыслов (аудитория)
clawhub install content-repurposer  # адаптация под платформы
clawhub install ai-humanizer      # убрать AI-пластик из текстов
```

## Установить всё одной командой

```bash
clawhub install copywriting editing headlines storytelling selling-meanings \
  shared-learnings telegram carousel reels youtube threads nano-banana \
  swipefile competitor-analysis news-summary seo-blog-writer ai-humanizer
```
