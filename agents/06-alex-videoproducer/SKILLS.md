# Скиллы — Алекс (Видео продюсер)

## Что устанавливать

```bash
clawhub install reels               # сценарии Reels/Shorts/TikTok
clawhub install youtube             # YouTube сценарии
clawhub install video-editor        # монтаж, субтитры, FCPXML
clawhub install veo-video           # генерация видео (Veo)
clawhub install timelapse-creator   # таймлапсы
clawhub install headlines           # хуки для видео
```

## Для veo-video
Нужен API-ключ Google. Добавить в `.env`:
```
GOOGLE_API_KEY=ваш_ключ
```
