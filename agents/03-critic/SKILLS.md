# Скиллы — Критик

## Что устанавливать

```bash
clawhub install editing             # инфостиль — главный инструмент критика
clawhub install copywriting         # понимание структуры постов
clawhub install shared-learnings    # антипаттерны и паттерны (обязательно)
```

## Как Критик работает
- Получает текст от Алёны
- Проверяет по чеклисту: хук / голос / клише / структура
- Возвращает ✅ или ❌ с конкретными правками
- Только после одобрения Критика текст идёт пользователю
