# Скиллы — Решала (CRM-бот)

Решала — публичный бот для потенциальных клиентов. Отдельная роль — не контент.

## Что устанавливать

```bash
# Решала не нуждается в контентных скиллах
# Его задача — встретить, рассказать, собрать заявку
```

## Настройка
Открой `SOUL.md` и замени:
- Название продукта
- Тарифы
- Контакт владельца (@твой_telegram)
- Описание команды

## Важно
- dmPolicy: open (отвечает всем, не только владельцу)
- Не устанавливай контентные скиллы — лишняя нагрузка
