# Скиллы — Решала (Координатор)

Решала координирует команду, управляет расписанием и задачами.

## Что устанавливать

```bash
clawhub install calendar      # управление календарём и встречами
clawhub install productivity  # планирование задач и приоритеты
```

## Важно
Решала НЕ создаёт контент — только координирует и делегирует.
Контентные скиллы не нужны.

## Схема делегирования
| Задача | Кому |
|--------|------|
| Стратегия, смыслы, распаковка | 🧠 Синсей |
| Текст, пост, статья | ✍️ Алёна |
| Карусель, картинка | 🎨 Маша |
| Аналитика, тренды | 📊 Максим |
| Reels, YouTube | 🎬 Алекс |
