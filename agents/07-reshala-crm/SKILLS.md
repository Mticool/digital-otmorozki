# Скиллы — Решала (Координатор)

Решала — главный. Принимает задачи и раздаёт команде.

## Что устанавливать

```bash
clawhub install selling-meanings    # понимает продающие смыслы — чтобы ставить правильные задачи
clawhub install prompt-engineer     # умеет формулировать задачи для агентов
```

## Важно
Решала сам не пишет контент — он раздаёт задачи правильным агентам.
Контентные скиллы не нужны — это нагрузка без смысла.

## Роль в команде
```
Владелец → Решала → Алёна / Маша / Максим / Алекс / Синсей
                 ↘ Критик (проверка всех текстов)
```
